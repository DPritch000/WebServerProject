# Exercise Tracker üí™

A simple, user-friendly web application to track your daily workouts and monitor your fitness progress.

## Features

- **Log Exercises**: Record exercise name, duration, calories burned, and date
- **View Statistics**: Track total exercises, total minutes worked out, and total calories burned
- **Persistent Storage**: Your exercise data is saved in the browser's local storage, so your data persists even after closing the browser
- **Delete Exercises**: Remove exercises from your log if needed
- **Responsive Design**: Works smoothly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and intuitive interface with a beautiful gradient design

## How to Use

1. **Open the Application**: Open `index.html` in your web browser
2. **Log an Exercise**: 
   - Fill in the exercise name (e.g., "Running", "Weight Lifting")
   - Enter duration in minutes
   - Add calories burned (optional)
   - Select the date
   - Click "Add Exercise"
3. **View Your Progress**: The statistics section updates automatically with your totals
4. **Manage Your Exercises**: View all logged exercises and delete any if needed

## Browser Requirements

- Modern web browser with JavaScript enabled
- Local Storage support (all modern browsers support this)

## How It Works

The application uses:
- **HTML5** for semantic structure
- **CSS3** for modern styling and responsive design
- **JavaScript ES6** with class-based architecture
- **Browser Local Storage** for data persistence

All data is stored locally in your browser and is never sent to any server.
